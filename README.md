# QuizMatic
***

[AppStoreにて公開中](https://apps.apple.com/jp/app/quizmatic/id1610312855?l=en)



◉コンセプト

英語で英語を学びたい人向けのクイズアプリです  


◉目的

日本語で英語を学ぶアプリは多くありましたが英語を英語で学べるものは無いと感じたので今回自分で作ってみました。

◉使い方

とてもシンプルな使い方です。

1: 初期画面でレベルを選択する

2: クイズが表示されるので正しい答えを選択する
